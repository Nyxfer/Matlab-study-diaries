## Basic Command

### Command

- `clc` clears the Command Window
- `clf` clears the Figure Window
- `clear` removes all variables from the Workspace
- `exist varname` checks if *varname* is already in use
- `help command` getting help


### symbol

- ` , `
- ` ; `
- ` : `


### operate window

- `CTRL + N` open new Script
